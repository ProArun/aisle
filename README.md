# Features implemented 

### 1)Login with registered number only like +919876543212
### 2)Session manegment by storing auth token into the shared pref database
### 3)Logout by deleting token
### 4)Ask user are they really wants to exit the aisle application.
### 5)Fecility to resend OTP if 60 second timer expired
### 6)To populate the user data used MVVM design pattern
### 7)Used View Binding to bind the view with kotlin class
